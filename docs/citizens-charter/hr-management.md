## 1. Request of Certificate of Employment / Service Record

- **Classification:** Simple
- **Type of transaction:** G2G, G2C
- **Who may avail:** City Employees

**Requirements → where to secure**

_Certification of Employment & Compensation:_

1. Request form duly accomplished by employees → Applicant
2. Employee's 201 file → HRMO
3. Payroll data/information → HRMO
4. System's information → HRMO

_Service Record:_

1. Request form duly accomplished by employees → Applicant
2. Employee's 201 file → HRMO
3. System's information → HRMO

**Fee:** None
**Processing time (total):** 3 days _(verify)_
