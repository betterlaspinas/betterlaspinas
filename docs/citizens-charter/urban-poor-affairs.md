## 1. Financial Assistance

- **Classification:** Highly Technical
- **Type of transaction:** G2C — Government to Citizen
- **Who may avail:** Informal Settlers affected of Demolition and Removal of Illegal Structures

**Requirements → where to secure**

_Filing of Application for Financial Assistance:_

1. Duly Accomplished Application Form (AF-001) with "2x2" ID Picture → Applicant
2. Marriage Contract/Birth Certificate (1 Certified True Copy and 2 Photocopies) → PSA
3. Certified True Copy and 2 Photocopies of Residency (1 Original and 2 Photocopies) → Applicant
4. At least 2 Valid ID's (3 Photocopies) → Applicant
5. Family Picture ("3x5") in front of the house/structure (1 Original Copy and 2 photographs) → Applicant
6. Lease/ Residence Certificate (Cedula) (1 Original Copy and 2 photographs) → Applicant
7. Picture of the house/structure before dismantling ("3x5") → Applicant

_Releasing of Check for Financial Assistance:_

1. Picture of the demolished house/structures (1 Original and 1 Photocopy) → Applicant
2. Both sides of 2 Valid ID's submitted during the application → Applicant
3. Signed and Notarized "Sinumpaang Salaysay" or Affidavit of Waiver → UPAO
4. UPAO Certification of demolished or dismantled structure → UPAO

| 3. Submission of requirements                                                         | Accept and review whether Application Form for Financial Assistance is accomplished and the requirements are complete (or otherwise it will not be accepted) | None | 5 minutes       | Janel D. Manulo     |
| 4. Wait for the Release of the financial assistance                                   | Preparation of DSR and Vouchers and other attachment/documents                                                                                               | None | 1 day           | Janel D. Manulo     |
|                                                                                       | Approval of the Office of the Mayor                                                                                                                          | None | 3 days          |                     |
|                                                                                       | Obligation Request to be signed by the Budget Office to verify there are available funds                                                                     | None | 3 days          |                     |
|                                                                                       | For review and checking of requirements by the City Accounting Office                                                                                        | None | 3 days          |                     |
|                                                                                       | Check Preparation, and signing at City Treasurer                                                                                                             | None | 3 days          |                     |
